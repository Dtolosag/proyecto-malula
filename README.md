# Proyecto Malula 🍞

Automatización para panadería artesanal de masa madre en Costa Rica.

## Descripción
Sistema de 3 agentes para automatizar:
- **Agente 1** — Coordinación con supermercado
- **Agente 2** — Pedidos por WhatsApp + pagos Stripe
- **Agente 3** — Mensajes de Instagram DM

## Stack
- n8n (automatización)
- Google Sheets (base de datos)
- Claude AI
- WhatsApp Business API / Meta Graph API
- Stripe
